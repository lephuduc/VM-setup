# Virtual-Machine
Quick setup for VM

## Usage

```bash
git clone https://github.com/lephuduc/VM-setup
cd VM-setup
chmod +x setup.sh
./setup.sh
```
## Python package
```bash
pip install -r requirements.txt
```
